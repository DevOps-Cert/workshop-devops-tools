## Starting Sonarqube

To start the container, run the following command on the folder of Sonarqube:

```shell
docker-compose up -d
```

## Stopping Sonarqube

To stop Sonarqube, simply run the following command on the Sonarqube folder:

```shell
docker-compose down
```
